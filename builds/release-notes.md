## 🎉 Initial Release

### Features
- 📦 Zip folders with automatic version numbering
- 🔢 Custom version input field (e.g., 1.0.1)
- ⏱️ Timestamp in filename
- 🌓 Dark mode support
- 📁 Built-in folder browser
- 🔄 Auto-increment versions

### Installation
1. Download FolderZipperVersioning.exe
2. Run the executable
3. Select a folder to zip
4. Optionally enter a custom version
5. Click 'Zip Selected Folder'

### File Naming
- Custom version: 1.0.1-foldername_2026-03-14_16-30.zip
- Auto-increment: foldername_001_2026-03-14_16-30.zip
