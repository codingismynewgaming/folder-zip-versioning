# Installation Guide  
  
## Prerequisites  
  
- Windows 10/11  
- Python 3.8+  
- pip 
  
## Steps  
  
1. Install Python from python.org  
2. Clone or download repository  
3. pip install -r requirements.txt  
4. python app-files/zipper.py  
  
## Build Executable  
  
pip install pyinstaller  
python build.py
